Design Justification

Impact of design & color choices

The palette (Rich Brown, Cream, Gold) evokes warmth and natural coffee tones, helping users associate the brand with artisanal roasting and quality. Typography pairing (Playfair Display for headings and Lato for body) gives an elegant café feel while remaining legible on small screens. Visual hierarchy is reinforced by generous spacing, card shadows, and rounded buttons to guide conversions (e.g., “Register” CTA).

Impact of HTML, CSS, JS, and Bootstrap on maintainability

We used semantic HTML5 tags (header, main, footer) for accessibility and easier content updates. Bootstrap 5 provides a responsive grid and ready components (navbar, carousel, accordion, table) which accelerates development and ensures consistent behavior across devices. A single external stylesheet (`styles.css`) holds theming variables (colors, fonts) so visual changes are centralized. JavaScript is lightweight and separated into `scripts.js` for maintainability — form validation logic, success/error messaging, and `localStorage` usage are modular and easy to extend.

Accessibility considerations
  • Alt attributes added for images and semantic form labels.
  • Contrast choices kept high between text and background for readability.
  • Form errors are shown in accessible alert containers.
Folder structure

Keeping all images in `/images`, CSS/JS files at project root helps graders and developers to locate resources quickly. Page filenames follow assignment naming conventions required by the brief.
